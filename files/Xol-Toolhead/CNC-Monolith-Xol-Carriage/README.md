# CNC Xol Monolith Compatible Carriages
## By MonkeyBonkey
These are 2 different carriages for Xol-toolhead or A4T designed for CNC milling. One of them is compatible with the universal Monolith belt clamp, the other one is made to be used with belt clamps that are on the sides, they're from another user mod.
